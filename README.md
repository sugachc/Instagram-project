# Instagram-project

<br>
## 🔥 프로젝트 목표
인스타그램 클론코딩 SNS 프로젝트

스프링부트+JPA      
AOP를 사용한 유효성 검사 자동화      
커스텀 예외 핸들링  
DTO매핑    
시큐리티     
OAuth2.0      
연관관계 처리     
다대다 처리     
ResponseEntity사용   


## ✨ 기술 스택

Spring boot,mysql,JPA,ajax,Spring security
